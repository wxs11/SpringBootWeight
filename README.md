# SpringBootWeight
Java 通过RS232 串口获取体重秤数据流 进行实时显示在前端进行校验

# 技术栈
* Springboot 
* Vue2 + ElementUi
* hutool
* Mybaitis-plus
* Sap
* RXTX

# 安装
将RXTX.jar 放入Java jre路径
* Copy RXTXcomm.jar ---> <JAVA_HOME>\jre\lib\ext
* Copy rxtxSerial.dll ---> <JAVA_HOME>\jre\bin
* Copy rxtxParallel.dll ---> <JAVA_HOME>\jre\bin
